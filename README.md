# 🎮 RPS R.A.N.D.O.M

A full-stack Rock Paper Scissors game built using Flask.

## 🚀 Features
- Random computer moves
- Score tracking
- Web UI with HTML, CSS, JS
- Reset functionality

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## ▶️ Run Locally
```bash
pip install flask
python app.py
